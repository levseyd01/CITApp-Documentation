View User Details#

The View User Details section allows you to both edit and view details of a specific user.

Navigating to the View User Details Section#

  1. Go to the homepage for either the User or Staff section.

  2. Click the View Icon next to the user you want to view:

  1. The View User Details page will open.

See Detailed Guide to User Details for more information about the View User Details section.
