## Accounts Tab: Table Columns

The {tab}`Accounts Tab` displays the following {column}`Columns`:

- {column}`Account Name`
- {column}`Type`
- {column}`Email`
- {column}`Phone`
- {column}`Action` ({action}`View Account`, {action}`Edit Permissions`, {action}`Delete`)
